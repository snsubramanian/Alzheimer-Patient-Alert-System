#Alzheimer patient

## Overview

SereneMind is a Django-based web application developed to support mental wellness and emotional well-being.

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite

## Project Structure

```text
SereneMind/
├── manage.py
├── README.md
├── requirements.txt
├── .gitignore
│
├── SereneMind/
│   ├── settings.py
│   ├── urls.py
│   ├── asgi.py
│   └── wsgi.py
│
├── SereneMindApp/
├── Templates/
└── Static/
```

## Installation

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
